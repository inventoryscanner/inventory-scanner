Setup guide

Complete these steps once to get fully set up.

1

Create the InventoryItems SharePoint list

Your SharePoint site → New → List → Blank list, name it InventoryItems. Add columns:

  • Barcode — Single line of text
  • ItemName — Single line of text
  • Quantity — Number
  • LowStockAt — Number
  • SupplierName — Single line of text
  • SupplierEmail — Single line of text
  • ReorderQty — Number
  • EmailTemplate — Multiple lines of text
  • NFCTagId — Single line of text (for NFC tag mapping)
2

Create the InventoryLog SharePoint list

Name it InventoryLog. Add columns: Action, Description (multi-line), UserName, Barcode, Quantity (Number).

3

Upload and configure

Upload inventory-scanner.html to SharePoint Documents, open it, go to Settings, paste your SharePoint site URL.

4

NFC tags — Scannable and others

The app reads NFC tags automatically on Android Chrome. When you tap a tag, it reads the NDEF payload — which could be:

  • Scannable tags: encode a URL like https://scannable.io/s/XXXXXX. The app extracts the serial/ID from the URL path and uses it as the barcode lookup key. Make sure the item's Barcode field in SharePoint matches that serial.
  • Plain text tags: encode a barcode string directly — these work as-is.
  • URL tags (non-Scannable): the full URL is used as the lookup key — set the item's Barcode field to match.

On iPhone: Apple doesn't support Web NFC in Safari, so use the Camera / QR mode instead.

On desktop: use the USB/Bluetooth barcode scanner.

5

Supplier email — Power Automate flow

Go to make.powerautomate.com → New flow → Instant cloud flow.

  1. Trigger: When an HTTP request is received
  2. Paste this JSON schema in the Request Body field:
{ "type":"object","properties":{
  "supplierEmail":{"type":"string"},
  "supplierName":{"type":"string"},
  "itemName":{"type":"string"},
  "barcode":{"type":"string"},
  "currentQty":{"type":"integer"},
  "reorderQty":{"type":"integer"},
  "emailBody":{"type":"string"},
  "sentBy":{"type":"string"}
}}
  1. Add action: Send an email (V2)
  2. To: triggerBody()?['supplierEmail']
  3. Subject: Reorder request — @{triggerBody()?['itemName']}
  4. Body: @{triggerBody()?['emailBody']}
  5. Save → copy the HTTP POST URL → paste in Settings → Supplier email
6

Teams webhook

In Teams channel → ··· → Connectors → Incoming Webhook → Configure → copy URL → paste in Settings.

7

Pin as a Teams tab

In Teams channel → + → Website → paste the SharePoint file URL → Save. All users share the same live inventory.
